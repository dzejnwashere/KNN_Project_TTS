# NeMo Guide
- clone NeMo from github: https://github.com/NVIDIA-NeMo/NeMo
- replace files as they are in `nemo` directory
  - example `nemo->colections->tts->losses->fastpitchloss.py` put to **NeMo/nemo/colections/tts/losses/fastpitchloss.py**
- do this to all files
- `conf` directory put the NeMo root path `NeMo/conf`
